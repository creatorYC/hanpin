# hanpin
> **hanpin** is a Chrome extension which can translate the selected Chinese words into pinyin.

### 项目依赖
- [pinyin](https://github.com/hotoo/pinyin)

### 特性
- 划词注音
- 自定义开启多音字
- 自定义开启分词

### 使用
本扩展没有发布到 Google 扩展中心，有需要的可以自己 clone 本项目，在chrome浏览器中以开发者模式安装，欢迎 star。
